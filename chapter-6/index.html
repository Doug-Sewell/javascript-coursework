Getting attributes and setting attributes

Getting attributes

Background color of this text is blue.

We can also update attributes. For example, this text will have a span tag in the HTML to set the text to have a background color of fuschia. However: This text will have a background color of lime green despite the HTML using fuschia. 


        //Get attribute:
        const orangeContent = document.querySelector('.orange-bg');
        console.log(orangeContent.getAttribute('style'));


        //Set attribute:
        document.querySelector('.background-to-change').setAttribute('style','background-color: limegreen');

Styling elements

This paragraph is styled within the HTML to be the color blue. However, using the JS below we styled it to be purple. (Assume we gave this paragraph a class of "purple-text".) We also set the font size to be 20 pixels.


        const purpleText = document.querySelector('.purple-text');
        purpleText.style.color = 'purple';
        purpleText.style.fontSize = '20px';
