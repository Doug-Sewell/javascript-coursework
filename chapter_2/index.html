JS Data Types

  • String => 1,2,3,100,3.14
  • Boolean => 'hello, world'

  • Undefined => Explicitly set variable with no value.
  • Object => Complex data structures such as Arrays, Dates, Literals, and more.
  • Symbol => Used with objects

Difference Between Functions & Methods

  • A function performs a specific task.
  • A method is a function associated with a particular object or data type.

Methods are called with dot notation.


//Method:
let result = name.toUpperCase();
console.log(result);

//Function:
const greet = () => 'hello';

let otherResult = greet();
console.log(otherResult);

Common String methods

  • string.toLowercase(); => returns the string in lowercase letters.
  • string.indexOf('@'); => returns the index within the string of the requested character.
  • string.lastIndexOf('@'); => Returns the index of the last instance of a requested character.
  • string.slice(0,5); => Slice takes in the starting index, and ending index of the portion of hte string you want.
  • string.substr(0,10); => Similar to slice, but this one takes in the starting position, and the second parameter is for how many spots you want to move forward. Not the index.
  • string.replace('m','w'); => Replaces the first instance of the supplied substring with the second parameter
  • someArray.includes('SomeValue'); => If the value is one of the values in an array, it will return true. (Otherwise, false.)

Common Array Methods

Let's pretend we have an array.
let someArray = [1, 5, 28, 2, 7];

  • let result = someArray.join(', '); would result in array elements being joined together as a string with a comma and space.
  • let num = someArray.indexOf(2); => Would set the value of num to 3 which is hte index of the number 2.
  • let otherArray = someArray.concat([20, 38, 30]); => would result in someArray with the digits 20, 38, and 30 added at the end.

While some array methods, like the ones above, do not alter the original arrays, some do. For example:

let wontPrintArray = someArray.push(82);
console.log(wontPrintArray); //This will print 6 which is the length of the array.
console.log(someArray); //This will print the updated array which is someArray from above with 82 at the end.

Methods that alter the original values are called destructive values.

And since we mentioned push(), let's do pop().

lastNum = someArray.pop(); //Now someArray has the original values and lastNum has a value of 82 which was added a few lines up.

Type Conversion

  • Convert a value to a number: Number("50");
    Note - If you put a String into the Number method, then it returns NaN.
  • Convert a number into a String: String(50);
  • Convert a value into a Boolean: Boolean(50);

Note you can get the type of value using the following syntax:
console.log(typeof someValue);

Truthy/Falsey Values
