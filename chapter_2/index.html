Chapter 2 Notes

JS Data Types

  • String => 1,2,3,100,3.14
  • Boolean => 'hello, world'
  • Null => true / false
  • Undefined => Explicitly set variable with no value.
  • Object => Complex data structures such as Arrays, Dates, Literals, and more.
  • Symbol => Used with objects

Difference Between Functions & Methods

  • A function performs a specific task.
  • A method is a function associated with a particular object or data type.

Common String methods

  • string.toLowercase(); => returns the string in lowercase letters.
  • string.indexOf('@'); => returns the index within the string of the requested character.
  • string.lastIndexOf('@'); => Returns the index of the last instance of a requested character.
  • string.slice(0,5); => Slice takes in the starting index, and ending index of the portion of hte string you want.
  • string.substr(0,10); => Similar to slice, but this one takes in the starting position, and the second parameter is for how many spots you want to move forward. Not the index.
  • string.replace('m','w'); => Replaces the first instance of the supplied substring with the second parameter
